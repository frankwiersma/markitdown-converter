API Usage

Convert using cURL:

curl -X POST \
    -F "file=@your_file.pdf" \
    http://localhost:8080/convert

# Or with URL:
curl -X POST \
    -F "url=https://example.com/document.pdf" \
    http://localhost:8080/convert

# For images with OpenAI analysis:
curl -X POST \
    -F "file=@image.jpg" \
    -F "api_key=your-openai-key" \
    http://localhost:8080/convert
